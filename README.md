# Reverb

Reverb is a modern social media iOS application built with SwiftUI, inspired by platforms like Twitter/X.
The project focuses on building a scalable architecture and clean UI while simulating real-world social app behavior.

## 🚀 Overview

Reverb is designed to explore how a social networking app works under the hood — including feed generation, user interaction, and modular UI design.

This project aims to demonstrate practical iOS development skills such as state management, architecture design, and preparing an app for real API integration.

## ✨ Features

* 📰 Social feed UI
* 👤 User profile interface
* ✍️ Post creation (UI/structure)
* ♻️ Reusable SwiftUI components
* ⚡️ Async-ready structure for future backend integration

## 🛠 Tech Stack

* Swift
* SwiftUI
* MVVM Architecture (partially implemented / in progress)

## 📂 Project Structure

* **Views** → UI components and screens
* **ViewModels** → State and business logic (in progress)
* **Models** → Data structures
* **Services** → Placeholder for API layer
